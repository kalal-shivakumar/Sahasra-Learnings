About Sahasra Learnings

Sahasra Learnings is a modern web application built with simplicity and scalability in mind. It demonstrates best practices in cloud deployment, CI/CD automation, and modern web development.

Technology Stack

Frontend: HTML5, CSS3, Vanilla JavaScript

Hosting: Microsoft Azure App Service (Linux)

CI/CD: GitHub Actions with OIDC Federation

Version Control: Git & GitHub

Project Goals

  • Demonstrate cloud-native deployment practices
  • Show modern authentication methods (Federated Credentials)
  • Provide a clean, responsive user interface
  • Implement Infrastructure as Code principles

Our Features

🎯 Automated Deployment

Every push to the main branch automatically triggers a build and deployment pipeline using GitHub Actions, ensuring continuous delivery of updates.

🔐 Security First

Uses OpenID Connect (OIDC) federated credentials instead of secrets, providing enhanced security and compliance with modern standards.

📱 Responsive Design

Built with mobile-first approach, ensuring the application works seamlessly across all devices and screen sizes.

⚡ High Performance

Optimized static content delivery on Azure CDN provides fast loading times and excellent performance metrics.
